# CLKG 数据采集 · 招募会速览

> 给新成员的第一页。5 分钟看完就知道要干什么。规则都在 Excel 模板里，不用背。

---

## 我们在做什么（一句话）

把分散在**文献、田野、地图、口述**里的文化遗产信息，整理成一个**统一、可查询、带出处**的知识库。
目前覆盖三个区域：**mustang**（木斯塘高原考古）、**qiaopi**（潮汕侨批）、**xinjiang**（新疆丝路遗产）。

## 招你来做什么（一句话）

用一个 Excel 模板，把资料按**"一行一个对象"**填进去。就这么简单。
**你不需要懂数据库**——填表就行，入库和技术的事负责人来做。

## 整个流程只有 3 步

```
①领一批资料  →  ②在模板里填行（照黄色示例填）  →  ③放进收件口、通知负责人
```

## 你会拿到一个 Excel：`CLKG_采集模板.xlsx`

打开它**第一个 sheet「填写说明」**，照着做即可。里面有 4 张采集表，**按对象选一张**：

| 你在记录…… | 填这张表 |
|---|---|
| 一处地点 / 一片区域 | **Place_地点**（单体填 pl，群体/区域填 clu） |
| 一份文档（侨批/档案/口述史） | **Document_文档**（doc） |
| 一个人 / 家族 / 机构 | **Actor_人物**（ac） |
| 一张照片 / 录音 / 录像 | **Asset_资产**（img） |

> 每张表第 2 行是黄色**示例行**，照着填，填完删掉。红色表头=必填，带下拉的列直接选。

## 只要记住 3 条红线

1. **坐标不自己换算**——高德填 `GCJ-02`、百度填 `BD-09`、GPS 填 `WGS84`，原样填+选对坐标系。
2. **同一个对象永远用同一个 `natural_key`**（它的"身份证"），填前先查权威清单，别重建。
3. **每行都要有来源**——文献填 Zotero 键、档案填馆藏号、网络填 URL。
